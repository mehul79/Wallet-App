* Clone Repo
```
git clone https://github.com/mehul79/Wallet-App
```
* Install dependencies for Backend
```
cd backend
npm i
npm start
```
* Install dependencies for Backend
```
cd frontend
npm i
npm run dev
````
* Go to the App
```
http://localhost:5173/
````
